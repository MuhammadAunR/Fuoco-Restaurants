# Fuòco Restaurant Website

A modern, elegant website built for **Fuòco** — a contemporary European and Italian fine dining restaurant with branches in Lahore and Islamabad, Pakistan.

---

## 🍽️ About Fuòco

Fuòco is one of Pakistan's premier fine dining destinations, offering a refined selection of contemporary European and Italian cuisine. With locations in Gulberg II, Lahore and F-6 Markaz, Islamabad, Fuòco delivers an exceptional dining experience rooted in quality, craft, and hospitality.

- **Instagram:** [@fuoco.restaurants](https://www.instagram.com/fuoco.restaurants/)
- **Lahore:** Gulberg II · 0305-2101111 · 1PM – 12AM
- **Islamabad:** F-6 Markaz · 0302-6333367 · 1PM – 1AM

---

## 🚀 Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** AOS (Animate on Scroll)
- **Icons:** Lucide React
- **Deployment:** Vercel
