# 🍣 Restaurant Website

## 📌 Overview
This is a modern, fully responsive restaurant website built using **Next.js**. It provides a seamless experience for users to browse the menu, place food orders, and make reservations. The project focuses on performance, smooth animations, and scalable state management.

---

## 🚀 Features

- 🛒 **Order Food System**
  - Browse categorized menu items
  - Add items to cart
  - Smooth ordering flow

- 📅 **Reservation System**
  - Easy table booking interface
  - User-friendly experience

- ⚡ **Smooth Animations**
  - Scroll-based animations using AOS
  - Smooth scrolling powered by Lenis

- 🔔 **Notifications**
  - Real-time alerts using react-hot-toast

- 🧠 **Global State Management**
  - Context API for managing cart and UI state

- 📱 **Fully Responsive**
  - Optimized for all screen sizes

---

## 🛠️ Tech Stack

- **Framework:** Next.js  
- **State Management:** Context API  
- **Animations:** AOS (Animate On Scroll)  
- **Smooth Scrolling:** Lenis  
- **Notifications:** react-hot-toast  
- **Styling:** Tailwind CSS
